///// Week 01 //////

ข้อที่ 1 Find the Shortest Word ให้หาคำที่สั้นที่สุดในประโยค โดยจะรับข้อความเข้ามา แล้วตัดออกมาเป็นคำๆ จากนั้นก็หาว่าคำไหนสั้นที่สุด 
ถ้ามีหลายคำที่สั้นเท่ากัน ก็ให้เอาคำแรกที่เจอ โดยจะนับเครื่องหมายวรรคตอนรวมเป็นส่วนหนึ่งของคำด้วย
ข้อที่ 2 Count Words in a Sentence ให้นับว่ามีคำทั้งหมดกี่คำในประโยค โดยคำจะถูกคั่นด้วยช่องว่าง 
และถ้ามีหลายช่องว่างติดกันก็ยังถือว่าคั่นคำอยู่ดี ถ้าข้อมูลที่ได้มาไม่ใช่ข้อความหรือเป็นข้อความว่าง ก็ให้ผลลัพธ์เป็นศูนย์
ข้อที่3 Reverse Words in a Sentence เป็นการสลับลำดับคำในประโยค ต้องให้คำสุดท้ายมาอยู่ข้างหน้า 
และคำแรกไปอยู่ข้างหลัง แต่ไม่ต้องสลับตัวอักษรในคำ แค่สลับตำแหน่งของคำ
ข้อที่4 Character Frequency Counter ข้อนี้ให้นับว่าตัวอักษรแต่ละตัวในข้อความว่ามีกี่ตัว นับทั้งตัวเล็กตัวใหญ่แยกกัน 
และนับทุกตัวรวมถึงช่องว่างด้วย ถ้าไม่มีข้อมูลหรือข้อมูลไม่ใช่ข้อความ ก็จะคืนค่ากลับเป็น {}
ข้อที่5 Check for Palindrome ให้ตรวจว่าข้อความที่รับมานั้นเป็น palindrome หรือไม่ ซึ่ง palindrome 
คือข้อความที่อ่านจากซ้ายไปขวาหรือขวาไปซ้ายแล้วได้เหมือนกัน Ex. madam และ racecar ถ้าไม่ใช่ข้อความหรือเป็นข้อความว่าง ก็จะคืนค่าเป็น falปse

ประโยชน์ที่จากการฝึก Practice

Homework Shallow Equality ในส่วนของ Object.keys(obj) นั้นจะคืนค่า array ของชื่อ key ใน object
และ ถ้าจำนวน key ไม่เท่ากัน จะคืนค่า false แล้วก็ ถ้าค่าใน key ไหนไม่ตรงกัน ก็จะคืนค่า false เหมือนกัน และนอกจากนี้
จะคืนค่าเป็น true

ข้อที่ 1 Student Object ให้สร้าง object นักเรียนที่เก็บข้อมูล โดยจะเก็บข้อมูลตามนี้ 1.id 2.name 3.subjects และให้เขียน method getSubjects 
เพื่อดึงรายวิชาออกมาเป็นข้อความ โดยจะการ join array
ข้อที่ 2 Constructor Function ให้สร้างฟังก์ชันแบบ constructor เพื่อสร้าง object นักเรียนที่มี id, name, major และจะมี method getProfile 
ที่คืนค่าเป็น string Ex. "1001 - Ann (IT)"
ข้อที่ 3 Class Car ให้สร้าง class ที่มี property brand และ model และมี method getCarInfo ไว้สำหรับคืนค่าข้อมูลรถ และจะมี method setModel 
ไว้เปลี่ยนค่า model ได้ด้วย
ข้อที่ 4 Object.create() ให้สร้าง object animal ที่มี property type และ method sound จากนั้นจะใช้ Object.create เพื่อสร้าง object dog 
เป็นการสืบทอดมาจาก animal และเพิ่ม property ใหม่พร้อมทั้ง override method sound ให้คืนค่าใหม่เป็น "Woof!"
ข้อที่ 5 Compare Book & Destructuring ให้สร้าง object หนังสือ 2 เล่มที่มีค่าเหมือนกัน จากนั้นจะเขียนฟังก์ชัน isSameBook 
เพื่อเปรียบเทียบว่า isbn เท่ากันมั้ย และใช้ destructuring ดึงค่า isbn และ title ของหนังสือออกมาด้วย

ประโยชน์จากการฝึก Practice

หรือการใช้ Object.create รวมถึงการใช้ destructuring และ shallow equality เพื่อตรวจสอบ object อีกด้วย

Homework Practice getFreqOfWords(sentence) จะตรวจสอบชนืดของข้อมูลก่อนเป็นอันดับแรกเพื่อป้องกันการเกิด error ฟังก์ชันจะคืนค่า undefined ถ้าพารามิเตอร์ไม่ใช่ string และต่อไปจะแยกคำของจากประโยค
โดยการใช้ .split() และสร้าง Object frequency จะเก็บคู่ key-value key ก็คือ คำ และ value = จำนวนครั้งที่ปรากฏ และในส่วนของลูป จะแปลงคำเป็น lowercase เพื่อรวมคำซ้ำต่าง case เช่น "Today" และ "today" 
และ (frequency[word] || 0) + 1 ด็คือถ้า key ยังไม่มีใน object ให้ใช้ค่า 0 และเพิ่มค่าทีละ 1

ข้อ 1 Function Declaration โดยสร้างฟังก์ชัน square(num) เพื่อคืนค่ากำลังสองของตัวเลข และสร้าง ฟังก์ชัน doSomething(fn) ที่รับฟังก์ชันเป็นพารามิเตอร์ แล้วเรียกฟังก์ชันนั้นด้วยค่า 7 หรือค่าอื่นๆ
ข้อ 2 Function Expression โดยสร้างฟังก์ชันแบบ Arrow Function ชื่อ divide รับเลข 2 ตัว และจะคืนค่าเป็นผลหาร
ข้อ 3 High Order Function โดยสร้างฟังก์ชัน sayHello() แล้วคืนค่า "Hello World" และจะสร้างฟังก์ชัน wrapper() คืนค่า ฟังก์ชัน sayHello และ ทดสอบเรียกโดยใช้ wrapper() จะคืนค่าฟังก์ชัน sayHello 
และ wrapper()() จะเรียกฟังก์ชันจริงโดยจะได้ Hello World ออกมา
ข้อ 4 Object Parameter Passing โจทย์จะกำหนด object มาให้ก็คือ car = { brand: "Toyota", year: 2020 } โดยจะให้เราสร้าง ฟังก์ชัน updateCar(obj) เปลี่ยน year เป็น 2025 
โดย Object ใน JavaScript จะส่งผ่าน reference
ข้อ 5 Spread และ Rest Operator ในส่วนของ SpreadOperator จะส่ง array [2, 4, 6] เข้าไปในฟังก์ชัน multiply3(a, b, c) และ ฟังก์ชัน multiply จะคืนค่าผลคูณ a * b * c
และในส่วนของ RestOperator ให้เขียนฟังก์ชัน sumAll(...nums) เพื่อรวมผลบวกของตัวเลขทั้งหมด และ ถ้าถามว่า spread กับ rest ต่างกันยังไง จำง่ายๆเลยว่า spread จะ แยกออก ส่วน rest จะรวมเข้ามา
