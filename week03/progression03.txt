ข้อ 1: Function Declaration
เขียนฟังก์ชัน square(num) ที่รับตัวเลข 1 ค่า แล้วคืนค่ากำลังสองของตัวเลขนั้น
จากนั้นเขียนฟังก์ชัน doSomething(fn) ที่รับฟังก์ชันเข้ามา แล้วให้ doSomething เรียกฟังก์ชัน fn ด้วยค่า 7

ข้อ 2: Function Expression (Arrow Function)
เขียนฟังก์ชันแบบ Arrow Function ชื่อ divide ที่รับตัวเลข 2 ค่า แล้วคืนค่าผลหาร

-----------------------------------------------------------------------------------------------

ข้อ 3: High Order Function
สร้างฟังก์ชัน sayHello ที่คืนค่า "Hello World"
และสร้างฟังก์ชัน wrapper() ที่จะ คืนค่าฟังก์ชัน sayHello

-----------------------------------------------------------------------------------------------

ข้อ 4: Object Parameter Passing
กำหนดให้มีออบเจ็กต์
let car = { brand: "Toyota", year: 2020 };
เขียนฟังก์ชัน updateCar(obj) ที่จะเปลี่ยนค่า year ให้เป็น 2025
แล้วลอง console.log ก่อนและหลังเรียกใช้

-----------------------------------------------------------------------------------------------

ข้อ 5: Spread และ Rest Operator
ใช้ Spread Operator ส่งอาร์เรย์ [2, 4, 6] เข้าไปในฟังก์ชัน multiply(a, b, c) ที่จะคืนค่า a * b * c
ใช้ Rest Operator เขียนฟังก์ชัน sumAll(...nums) ที่รวมผลบวกของเลขทุกตัว
